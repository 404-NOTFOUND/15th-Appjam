## # How did we develop?

- Python의 Flask 프레임워크를 이용한 WebApp을 제작한 뒤 Android의 Web View 애플리케이션과 연동

- 웹앱의 데이터베이스 접근을 위해서 SQLAlchemy라는 툴킷을 이용함

- Tensorflow와 Inception-v3 모델을 이용한 AI를 사용해서 자동으로 컨텐츠에 맞는 태그를 다는 기능을 구현
